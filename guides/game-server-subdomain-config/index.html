Configure Subdomains for Game Servers and Multiplayer

Use SRV records to point game server subdomains to non-standard ports, configure DNS for Minecraft, Valheim, and other game servers hosted on residential or dedicated servers.

Overview

Game servers often run on non-standard ports (25565 for Minecraft, 2456 for Valheim). SRV records let you use a clean subdomain like mc.yourname.is-pro.dev without players needing to remember port numbers.

Prerequisites

  • A game server running on a known IP and port
  • An is-pro.dev subdomain registered at dash.is-pro.dev
  • Basic understanding of SRV record syntax

Step 1: Set Up Your Game Server

Ensure your game server is running and reachable. For home-hosted servers, forward the game port (e.g. 25565 for Minecraft) in your router's settings. Verify external access using a tool like canyouseeme.org — if the port is blocked, use Cloudflare Tunnel instead.

Step 2: Configure the SRV Record on is-pro.dev

Add an SRV record on the is-pro.dev dashboard. The format is: _service._protocol.subdomain TTL IN SRV priority weight port target. For Minecraft this looks like: _minecraft._tcp 3600 IN SRV 0 5 25565 mc.yourname.is-pro.dev. The _tcp prefix is standard — most games use TCP.

Step 3: Add a Fallback A Record

Some clients or launchers do not support SRV lookups. Add an A record pointing directly to your server IP as a fallback. Set a low TTL (60 seconds) so you can change IPs quickly if needed.

Step 4: Configure Firewall and Security

Limit connections to the game port only. Use iptables or UFW on Linux: ufw allow 25565/tcp. For DDoS protection, enable Cloudflare proxy (orange cloud) — note that this only works for HTTP-based games. TCP games require DNS-only mode (gray cloud).

Best Practices

  • Use descriptive subdomains like mc, ark, or valheim for each server
  • Set SRV TTL to 300–600 seconds for standard setups, lower during IP changes
  • Test SRV resolution with dig _minecraft._tcp.yourname.is-pro.dev SRV
  • Use Cloudflare Spectrum for TCP/UDP game proxy (paid plan required)

FAQ

Do all games support SRV records?

Most games with direct-connect functionality support SRV (Minecraft, Valheim, Source games). Check your game's documentation or try adding the IP:port manually if SRV doesn't work.

Can I use SRV with Cloudflare proxy enabled?

No — Cloudflare proxy only works for HTTP/HTTPS traffic. For game servers, use DNS-only mode (gray cloud) and rely on Cloudflare's DNS for fast resolution.
