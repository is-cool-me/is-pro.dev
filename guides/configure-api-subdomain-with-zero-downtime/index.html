Guide: Configure an API Subdomain with Zero Downtime

Migrate API traffic by lowering TTL and swapping targets safely.

Implementation steps

  • Lower TTL at least one full cache cycle before migration.
  • Deploy the new API endpoint and validate health checks.
  • Switch DNS target during a low-traffic window.
  • Monitor errors and latency, then restore normal TTL after stability.

Validation checklist

  • Authoritative DNS answers return expected values.
  • Target platform verifies the custom domain.
  • HTTPS is active and redirect behavior is correct.
  • No stale records remain from previous setup.
