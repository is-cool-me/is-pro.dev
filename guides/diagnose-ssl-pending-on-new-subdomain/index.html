Guide: Diagnose SSL Pending on a New Subdomain

Common certificate issuance blockers and quick fixes.

Implementation steps

  • Confirm DNS resolves correctly from authoritative nameservers.
  • Check whether proxy mode or origin certificate config blocks issuance.
  • Verify the host serves expected responses on HTTP and HTTPS.
  • Re-run certificate issuance after fixing record or origin mismatches.

Validation checklist

  • Authoritative DNS answers return expected values.
  • Target platform verifies the custom domain.
  • HTTPS is active and redirect behavior is correct.
  • No stale records remain from previous setup.
