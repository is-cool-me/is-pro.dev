Connect Shopify Store to a Custom Subdomain

Point a subdomain from is-pro.dev to your Shopify store, configure DNS records for Shopifys CDN, and set up proper SSL to accept payments on your branded store URL.

Overview

Shopify allows one custom domain per store. By using an is-pro.dev subdomain, you get a branded URL without the $14/year domain fee. This guide covers DNS setup and SSL configuration for payment acceptance.

Prerequisites

  • A Shopify store (Basic plan or higher supports custom domains)
  • An is-pro.dev subdomain registered at dash.is-pro.dev
  • Access to your Shopify admin panel

Step 1: Add Domain in Shopify Admin

Go to Shopify Admin → Settings → Domains → "Add domain". Enter your is-pro.dev subdomain (e.g. store.yourname.is-pro.dev). Shopify shows the required DNS configuration — typically a CNAME record pointing to shops.myshopify.com.

Step 2: Configure DNS on is-pro.dev

Add a CNAME record on the is-pro.dev dashboard targeting shops.myshopify.com. Do not enable Cloudflare proxy (gray cloud) for Shopify — Shopify handles its own SSL and proxying through Cloudflare can interfere with payment processing and certificate verification.

Step 3: Verify Domain in Shopify

Back in Shopify, click "Verify connection". Shopify checks the DNS record and, if correct, marks the domain as connected. SSL certificate provisioning through Shopify's Let's Encrypt integration starts automatically. This can take up to 48 hours but usually completes within 1 hour.

Step 4: Set Primary Domain

Once verified and SSL is active, set your is-pro.dev subdomain as the primary domain in Shopify settings. All traffic will redirect from the default yourstore.myshopify.com to your custom subdomain. Existing product links and SEO rankings are preserved automatically.

Best Practices

  • Keep Cloudflare proxy (orange cloud) disabled — gray cloud only for Shopify
  • Do not set up a 301 redirect from your subdomain to the myshopify.com URL
  • Test checkout flow end-to-end before announcing the new domain
  • Update your Facebook Pixel and Google Analytics domains after switching

FAQ

Will changing my domain affect SEO?

Shopify handles the migration automatically with 301 redirects from your old myshopify.com URL. Search rankings are preserved as long as the old URL remains active for a few weeks.

Can I use Cloudflare proxy for my Shopify store?

Not recommended — Cloudflare proxy can interfere with Shopify's certificate verification and payment gateway callbacks. Use DNS-only (gray cloud) mode.
